# Full-Stack Kubernetes Deployment Project

## 🚀 Overview
This project demonstrates a full-stack application deployed using Docker and Kubernetes.

## 🧩 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Containerization: Docker
- Orchestration: Kubernetes

## 🔧 Features
- Frontend interacts with backend API
- Backend provides REST API
- Dockerized services
- Kubernetes Deployment & Services
- Frontend ↔ Backend communication

## 📁 Project Structure
backend/
frontend/
k8s/


## ⚙️ How to Run:
### 1. Build Docker Images
### 2. Deploy to Kubernetes
### 3. Access Application

## 🐳 Docker
- Created Dockerfiles for frontend and backend
- Built images using docker build
- Used containers for deployment

Then open:
http://localhost:3000

## 🎯 Output
Frontend fetches message from backend successfully.
